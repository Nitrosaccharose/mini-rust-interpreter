## 🧭 项目简介
本项目使用Rust语言实现了一个简单的解释器，并提供对应调用的API接口，使得前端界面可以使用。
<br/>
本项目是为了初学Rust语法进行的练习，所以代码可能存在Bug，欢迎大家提出Issues！

## 🍔 食用方法
使用以下命令运行后端：
```
cd backend_rust
cargo build
.\target\release\backend_rust.exe
```
<br/>
使用以下命令运行前端：

```
cd go-web-front-project
yarn install
yarn dev
```
